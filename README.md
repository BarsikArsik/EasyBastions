# EasyBastions
A module for Foundry VTT that allows you to build a bastion in DND2024 using a new tool for transferring scenes as tiles while preserving all scene components (walls, lighting, tokens, etc.). Scene presets for construction are included in the module.
***
Модуль для Foundry VTT, позволяющий строить бастион в DnD2024, с помощью нового инструмента переноса сцен как тайлов с сохранением всех компонентов сцены (стены, освещение, токены и т.д.). Пресеты сцен для строительства включены в модуль. 

# Как это работает
В модуле добавлены Основные строения и Особые строения из книги правил DnD2024. С выставленными стенами, освещением и актерами. *А так же вспомогательные ресурсы вроде Журнала по Бастионам и все случайные таблицы*. Вы не ограничены тем, что есть в модуле и можете добавить любые новые сцены по своему желанию!
Сейчас в модуле присутствуют Особые строения 5 уровня. В дальнейшем планируется расширение до 17 уровня.

Вы можете перетащить сцену из боковой панели или из справочника на холст; после этого появится тайл с фоновым изображением исходной сцены. Вы можете перемещать и вращать тайл, чтобы расположить его по своему усмотрению.

После установки тайла нажмите кнопку блокировки (пкм на тайл, значок замка) на HUD-панели плитки. После блокировки все объекты из исходной сцены, включая токены, стены, источники света и т. д., появятся поверх тайла. Если вам нужно изменить положение тайла, просто разблокируйте его, что удалит все объекты, после чего вы можете переместить тайл и снова его заблокировать.
**NB! В Foundry V14 изменилась логика работы с объектами, поэтому чтобы разблокировать тайл вам надо  в правой части экрана найти раздел объекты, выбрать там нужный тайл и разблокировать его. (кнопка Alt подсветит все тайлы на сцене)**
